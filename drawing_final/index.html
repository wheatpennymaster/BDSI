Muller Diagram Generator v2

This is a Java Applet that works in browser. It's pretty solid. Problems include incorrectly representing the frequency of mutations, incorrectly plotting mutations that fail to fix or go extinct at the last generation, and the fact that it is an applet. Your browser will warn you about security issues with running the applet, but the only thing it does it access the computer's clipboard so that you're able to paste in data rather than type it.

Muller Diagram Generator v3

This is a Java Applet that works in browser. It's pretty solid. It solves some of the issues of the previous applet but has new problems. Sometimes the curves overlap where they should not. Again, this is an applet, so there will be a security warning generated by the browser. It only will access your computer's clipboard.

Muller Diagram Generator v4

Much improved from the previous version. Completely different method for determining how the curves should be drawn. Shows competition between two mutations by having them "touch" one another. Also improved in that it uses an HTML form for choosing options. The result is more security. The data uploaded will be saved temporarily on the server and deleted once the session is over.

Muller Diagram Generator v5

About the same as the previous version. It's hard to get everything working at once, so this version does some things better and some things worse thant v4. The PHP also gives you a preview of the image in the browser as you download it. It will delete the image off the server once the session is over.

Muller Diagram Generator v6

This version handles how the starting generation is determined. Essentially, the program takes the first two data points above a certain slope and estimates the point at which the frequency is zero. Also this version has a different algorithm for placing the names of the mutations in the Muller diagram.
